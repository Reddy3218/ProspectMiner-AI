# ProspectMiner AI Architecture

## Overview
ProspectMiner AI is a MERN-based lead mining platform that scrapes business data,
enriches it using AI, and ranks leads based on quality.

## Components

### Backend
- Node.js + Express API
- MongoDB for data storage
- Puppeteer for scraping business listings
- Claude AI for lead enrichment

### Frontend
- React dashboard
- Campaign tracking UI
- Lead management interface

## Workflow

1. User creates a search campaign
2. Scraper collects business data
3. AI analyzes business websites
4. Leads are scored and ranked
5. User exports leads as CSV

## Future Improvements

- LinkedIn enrichment
